## Development

### Prerequisites

1. Node >= 16.13.0

### Getting Started

1. Use correct version of Node: `nvm use`
1. Review instructions in `./vsc-extension-quickstart.md` to enable live preview
1. Edit theme defined in `./themes/Dessert-color-theme.json`
